SVG Percentage

i

This sample was an excuse for me to check out a new library: snapsvg. The Percentage object is configurable with options to supply a percentage endpoint, duration and easing.

Block Toggles

i

This sample includes a couple of handy scripts for the display of content. Content is displayed in a blocks object which is a percentage based list. An equalHeight script is applied to maintain the grid-like structure of the elements while a separate script handles the visibility toggleing of content items.
